# Separation (FashionMNIST classification)
Ограничение времени	-- 20 секунд
Ограничение памяти	-- 256 Мб
Ввод	-- стандартный ввод или no input
Вывод	-- стандартный вывод или tests.log

Решите задачу Separation (hw_fmnist) репозитории в папке homeworks/hw01_classification. После решения задачи классификации сгенерируйте посылку и загрузите в данную задачу submission_dict_task_1.json. Для этого файл hw**_data_dict.json из папки с задачей должен находиться в той же директории, что и ноутбук. Скрипт для автоматической загрузки доступен в ноутбуке, но при возникновении ошибок попробуйте скачать файл вручную.

Сгенерированный объект отправьте для сдачи в контест.
